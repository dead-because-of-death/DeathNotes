# DeathNotes
Web app for saving notes and dictionaries that could be shared with friends.

Using :
Java,
Maven,
Spring(Hibernate(JPA framework), Spring Security, Spring boot(Apache Tomcat inside), Thymeleaf),
Lombok,
H2 database.
